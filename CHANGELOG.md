# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-06-16
### Added
- **Interactive Map System:** Authors can now create world maps and place markers for locations (Cities, Dungeons, etc.).
- **Multi-Language Support:** Native typography and support for Bengali and Hindi.
- **Reading Streak Engine:** Persistent tracking of daily reading habits with streak logic.
- **Professional Studio Upgrade:** Integrated Tiptap Bubble Menu and "Story Bible" sidebars.
- **Analytics Heatmap:** GitHub-style activity visualizer on the User Dashboard.

## [1.1.0] - 2026-06-15
### Added
- **Writing Studio:** First iteration of the distraction-free editor with auto-save.
- **Immersive Reader:** Initial release of the reading interface with theme support.
- **Story Detail Page:** Dynamic cover hero and chapter management.
- **Library Module:** Collection management and "Continue Reading" logic.

## [1.0.0] - 2026-06-14
### Added
- **Modular DDD Architecture:** Established the core directory structure.
- **Animated Auth:** Custom GSAP-powered Rube Goldberg login/registration system.
- **Premium Dashboards:** Specialized views for both Admin and standard Users.
- **Search System:** Ctrl+K command pallet with live backend results.
- **Database Schema:** Core tables for Stories, Chapters, Genres, and Users.

### Fixed
- Improved mobile responsiveness for the animated auth machine.
- Resolved Tailwind v4 scanning issues for modular Blade files.
